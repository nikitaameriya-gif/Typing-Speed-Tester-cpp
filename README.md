# Typing Speed Tester (C++ Pro Version)

## Description
A console-based typing speed tester that measures WPM and accuracy with multiple difficulty levels and leaderboard functionality.

## Features
- Difficulty levels (Easy / Medium / Hard)
- WPM calculation
- Accuracy calculation
- Leaderboard (file storage)
- Menu-driven system

## Concepts Used
- OOP (Classes)
- File Handling
- String Processing
- Chrono Library (Time measurement)
- Menu-driven programming

## Run
```bash
g++ main.cpp TypingTest.cpp ScoreManager.cpp -o typing
./typing
